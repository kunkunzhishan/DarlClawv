# DarlClawv

DarlClawv is a simple, Codex-first assistant.

It focuses on one thing: helping you run tasks quickly with a clean local workflow.

Inspired by OpenClaw, which set a strong example for practical and elegant agent design.

## Quick Start

1. Install deps: `npm install`
2. Build: `npm run build`
3. Run task (blocking until final output): `node dist/src/cli/index.js run --task "hello"`
